# We Masau Loan Portal

A complete loan application management system for We Masau SACCO.

## Features

- ✅ Loan application submission
- ✅ Multi-approval workflow (3 committee members required)
- ✅ Google Sheets integration for data storage
- ✅ Email notifications for committee members
- ✅ Admin dashboard for approvals
- ✅ CSV export functionality
- ✅ Mobile responsive design

## Setup Instructions

1. Deploy the Google Apps Script backend
2. Update the `APPS_SCRIPT_URL` in index.html
3. Host on GitHub Pages
